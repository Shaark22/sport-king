## Запуск на локал хосте (у себя на пк)

```
npm install
npm run dev
```

сайт откроется http://localhost:5173

админка - http://localhost:5173/sk-manage-kz8m2p/login  
пароль в .env файле ADMIN_PASSWORD (локально можно sportking)

если надо собрать перед сервером:
```
npm run build
npm start
```

проверить что ничего не сломалось:
```
npm run lint
npm run test:api
npm run qa
```
qa это когда dev уже запущен - сам сайт в edge откроет и прогонит

## что есть на сайте

каталог товаров, отзывы, kaspi кнопки  
есть заявка "купить на сайте" и whatsapp  
страницы о нас доставка контакты  
**фотогалерея** - фото от клиентов, заливаешь из админки  
hero и тексты и телефоны тоже в админке меняются

## админка

секретный путь /sk-manage-kz8m2p/login (не /admin - он выключен на проде)

там можно:
- товары - добавить менять цены фото, галочка в наличии
- категории
- отзывы
- заявки с сайта
- **фотогалерея** - загрузка фото клиентов, можно скрыть с сайта если надо
- настройки - hero контакты инста тексты страниц
- статистика - кто заходил клики kaspi контакты
- ошибки - если сервер что то выдал

5 раз не тот пароль - блок 15 минут

данные в server/data/ store.json settings.json gallery.json uploads/

## фотогалерея

админка -> фотогалерея -> загрузить  
на сайте только опубликованные  
фото сжимаются перед загрузкой чтоб сервер не лег  
лучше не заливать пачкой по 50 штук сразу

## фото товаров

советую для карточек **вставлять ссылку на фото** (kaspi cdn и тд) - так меньше грузит сервер когда товаров много  
файлом на сервер - для hero и своих картинок

## в наличии
